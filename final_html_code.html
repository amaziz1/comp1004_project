Welcome to the SPA Project: TaskFlow

This is an interactive Single Page Application (SPA) to manage your daily tasks with dynamic content and smooth navigation.

About This Project

This SPA was developed as a project for the COMP1004 course. It demonstrates dynamic content loading, a task manager, and an interactive interface with dark/light mode.

Features

  • Interactive Task Management
  • Dynamic Dark Mode Toggle
  • JSON / CSV Import/Export for tasks
  • Smooth Navigation
  • Task Prioritization and Progress Tracker
  • Task Reminders
  • Multiple Lists / Projects Support
  • Recurring Tasks
  • Task Time Tracking
