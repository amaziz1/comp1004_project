Welcome to the SPA Prototype

This is an interactive Single Page Application (SPA) to manage your daily tasks with dynamic content and smooth navigation.

About This Project

This SPA was developed as a project for the COMP1004 course. It demonstrates dynamic content loading, task management, and an interactive interface with dark/light mode, inverted colour mode, and gamification features.

Features

  • Interactive Task Management
  • Dark/Light Mode Toggle
  • Inverted Colour Mode
  • Prioritization and Deadlines
  • Task Categories/Tags
  • Collaboration (Coming Soon)
  • Reminders and Notifications
  • Gamification (Badges and Streaks)
